# Keyboard Shortcuts

All keyboard shortcuts available in Mac Leather Dashboard.

---

## Global Shortcuts

Available on any page.

| Shortcut | Action |
|----------|--------|
| `Ctrl/Cmd + K` | Open global search |
| `Ctrl/Cmd + /` | Open keyboard shortcuts help |
| `Esc` | Close modal / Cancel action |
| `Ctrl/Cmd + S` | Save current form |
| `Ctrl/Cmd + Enter` | Submit current form |

---

## Navigation Shortcuts

| Shortcut | Action |
|----------|--------|
| `G` then `H` | Go to Home / Dashboard |
| `G` then `S` | Go to Samples |
| `G` then `P` | Go to Production |
| `G` then `E` | Go to Exports |
| `G` then `O` | Go to Procurement (Orders) |
| `G` then `U` | Go to Suppliers |

---

## Search Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl/Cmd + K` | Open global search |
| `/` | Focus search input (on list pages) |
| `Enter` | Execute search / Select result |
| `↑` / `↓` | Navigate search results |
| `Esc` | Clear search / Close search |

### Search Tips

| Prefix | Searches |
|--------|----------|
| `SMP-` | Samples by code |
| `PRD-` | Production orders by code |
| `PO-` | Purchase orders by code |
| `SHP-` | Shipments by code |
| `JC-` | Job cards by code |

---

## List Page Shortcuts

Available on pages with tables/lists.

| Shortcut | Action |
|----------|--------|
| `N` | Create new record |
| `R` | Refresh list |
| `/` | Focus search/filter |
| `↑` / `↓` | Navigate rows |
| `Enter` | Open selected row |
| `Ctrl/Cmd + Click` | Open in new tab |

---

## Detail Page Shortcuts

Available on detail/edit pages.

| Shortcut | Action |
|----------|--------|
| `E` | Edit record |
| `Ctrl/Cmd + S` | Save changes |
| `Esc` | Cancel edit / Close modal |
| `D` | Delete record (if allowed) |
| `Ctrl/Cmd + P` | Print page |

## Browser Shortcuts

|----------|--------|

## Platform Differences

| Action | Windows/Linux | Mac |
|--------|---------------|-----|
| Primary modifier | `Ctrl` | `Cmd (⌘)` |
| Save | `Ctrl + S` | `Cmd + S` |
| Search | `Ctrl + K` | `Cmd + K` |
| New tab | `Ctrl + Click` | `Cmd + Click` |
| Print | `Ctrl + P` | `Cmd + P` |
