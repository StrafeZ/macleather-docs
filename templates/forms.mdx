## Troubleshooting

<AccordionGroup>
  <Accordion title="Form fields don't match system">
    **Cause:** Template outdated.

    **Solution:** Download latest version from this page.
  </Accordion>

  <Accordion title="Can't edit PDF form">
    **Cause:** PDF is not fillable.

    **Solution:** Print and complete manually, or use XLSX version if available.
  </Accordion>

  <Accordion title="Excel formulas not working">
    **Cause:** Protected cells or macros disabled.

    **Solution:** Enable editing when prompted. Check macro settings.
  </Accordion>
</AccordionGroup>
