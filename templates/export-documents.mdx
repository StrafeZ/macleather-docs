| Total Value | Line totals |

## Troubleshooting

<AccordionGroup>
  <Accordion title="Template won't open">
    **Cause:** Wrong software version.

    **Solution:** XLSX requires Excel 2007+ or Google Sheets. PDF requires any PDF reader.
  </Accordion>

  <Accordion title="Fields not matching system">
    **Cause:** Template outdated.

    **Solution:** Download latest version from this page.
  </Accordion>

  <Accordion title="Formatting issues when printing">
    **Cause:** Printer settings incorrect.

    **Solution:** Set to A4, Portrait, 100% scale.
  </Accordion>
</AccordionGroup>
