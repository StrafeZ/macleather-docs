## Template Formats

| Format | Use For |
|--------|---------|
| PDF | Print-ready documents |
| XLSX | Editable spreadsheets |
| ZPL | Zebra label printers |
| PNG | Image-based labels |

---

## How to Use Templates

<Steps>
  <Step title="Download template">
    Click download link on template page
  </Step>
  <Step title="Fill in data">
    Replace placeholder text with actual values
  </Step>
  <Step title="Print or upload">
    Print directly or upload to system
  </Step>
</Steps>

---

## System-Generated vs Manual

| Type | When to Use |
|------|-------------|
| System-generated | Preferred - auto-fills from order data |
| Manual templates | Backup when system unavailable |

<Info>
Most documents can be auto-generated from the system. Use manual templates only when necessary.
</Info>
