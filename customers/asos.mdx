# ASOS Requirements

This guide covers requirements for ASOS shipments, including the mandatory ASN (Advance Shipment Notification) process.

<Warning>
ASOS requires ASN submission before shipment departure. Shipments without valid ASN will be refused at the DC.
</Warning>

---

## ASN (Advance Shipment Notification)

### What is ASN?

ASN is an electronic notification sent to ASOS before your shipment departs. It tells ASOS exactly what's coming so they can prepare to receive it.

### ASN Requirement

| Requirement | Specification |
|-------------|---------------|
| Mandatory | Yes - no exceptions |
| Timing | Submit 24-48 hours before departure |
| Format | EDI or ASOS Supplier Portal |
| Confirmation | Wait for ASN acknowledgment |

### ASN Process

<Steps>
  <Step title="Prepare shipment data">
    Carton contents, weights, dimensions finalized
  </Step>
  <Step title="Generate ASN in system">
    Exports → Shipment → Generate ASN
  </Step>
  <Step title="Submit to ASOS portal">
    Upload or transmit via EDI
  </Step>
  <Step title="Receive confirmation">
    Wait for ASOS acknowledgment email
  </Step>
  <Step title="Ship goods">
    Only after ASN confirmed
  </Step>
</Steps>

### ASN Data Required

| Field | Description |
|-------|-------------|
| Shipment Reference | Your shipment number (SHP-YYYY-XXX) |
| PO Numbers | All POs included in shipment |
| Carton Count | Total number of cartons |
| Carton Details | SSCC, contents, weight per carton |
| Ship Date | Actual departure date |
| Carrier | Shipping company name |
| Tracking | AWB or tracking number |
| ETA | Expected arrival at DC |

<Info>
The system can auto-generate ASN data. Go to Shipment Detail → Documents → Generate ASN.
</Info>


### Label Elements

| Element | Position | Format/Requirement |
|---------|----------|-------------------|
| ASOS Logo | Top center | ASOS text logo |
| Supplier Name | Upper section | Your company name |
| PO Number | Upper section | ASOS PO reference |
| SKU | Middle section | ASOS SKU number |
| Description | Middle section | Product description |
| Sizes | Middle section | All sizes in carton |
| Total Quantity | Middle section | Total units |
| SSCC Barcode | Lower section | 18-digit SSCC |
| SSCC Number | Below barcode | Human-readable |
| Carton Number | Bottom | "X of Y" format |
| Weight | Bottom | Gross weight in kg |

### Label Specifications

| Specification | Requirement |
|---------------|-------------|
| Size | 100mm × 150mm |
| Orientation | Landscape |
| Paper | White thermal, semi-gloss |
| Barcode Type | Code 128 |
| Barcode Height | Minimum 25mm |
| Quiet Zone | Minimum 10mm each side |
| Print Quality | Grade B or better |
| Placement | One label on longest side |

## Packing Requirements

### Carton Specifications

| Specification | Requirement |
|---------------|-------------|
| Maximum Weight | 20 kg gross |
| Maximum Length | 70 cm |
| Maximum Width | 50 cm |
| Maximum Height | 50 cm |
| Carton Material | Single or double-wall corrugated |
| Sealing | Brown tape, I-seal acceptable |

### Packing Rules

<AccordionGroup>
  <Accordion title="Size Sorting">
    **Rule:** Mixed sizes allowed in same carton.

    ASOS accepts mixed size cartons, unlike some other customers.

    | Allowed | Example |
    |---------|---------|
    | Mixed sizes | 4 × Size 8, 4 × Size 10, 4 × Size 12 |
    | Single size | 12 × Size 10 |
    | Mixed styles | Not allowed - single style per carton |

    **Important:** Even though mixed sizes are allowed, each carton must contain only ONE style/SKU.
  </Accordion>

  <Accordion title="Poly Bagging">
    **Rule:** Individual poly bagging required.

    | Specification | Requirement |
    |---------------|-------------|
    | Bag Material | Clear recyclable polypropylene |
    | Bag Thickness | Minimum 30 micron |
    | Closure | Self-seal |
    | Recycling Mark | Mobius loop symbol required |
    | Warning | Suffocation warning in 3 languages |

    **Languages for warning:** English, French, German
  </Accordion>

  <Accordion title="Sealing Method">
    **Rule:** I-seal pattern acceptable.

    ```
    ┌───────────────┐
    │ ═══════════   │ ← Single top seal
    │               │
    │               │
    │               │
    │ ═══════════   │ ← Single bottom seal
    └───────────────┘
    ```

    | Specification | Requirement |
    |---------------|-------------|
    | Tape Width | 50mm brown packing tape |
    | Top Seal | Full width across center of top |
    | Bottom Seal | Full width across center of bottom |
    | No staples | Staples not permitted |
  </Accordion>

  <Accordion title="Sustainability Requirements">
    **Rule:** ASOS has specific sustainability requirements.

    | Item | Requirement |
    |------|-------------|
    | Poly Bags | Must be recyclable |
    | Cartons | Recyclable cardboard |
    | Void Fill | Paper only (no plastic) |
    | Tape | Paper tape preferred |
    | Labels | FSC certified paper preferred |
  </Accordion>
</AccordionGroup>

## Troubleshooting

<AccordionGroup>
  <Accordion title="ASN rejected">
    **Common Causes:**
    - Data mismatch with PO
    - Invalid SSCC codes
    - Missing required fields

    **Prevention:**
    1. Verify all PO numbers are correct
    2. Generate SSCC from system (don't create manually)
    3. Check all required fields before submission
    4. Validate against ASOS specifications
  </Accordion>

  <Accordion title="Shipment refused at DC">
    **Cause:** No valid ASN or ASN not acknowledged.

    **Prevention:**
    1. Always wait for ASN acknowledgment
    2. Keep confirmation email/printout
    3. Include ASN reference in shipping documents
    4. Never ship without confirmation
  </Accordion>

  <Accordion title="SSCC barcode not scanning">
    **Cause:** Incorrect SSCC format or print quality.

    **Prevention:**
    1. Use system-generated SSCC codes
    2. SSCC must be exactly 18 digits
    3. Include check digit
    4. Test scan before applying to carton
  </Accordion>

  <Accordion title="Carton rejected for mixed styles">
    **Cause:** Multiple SKUs in same carton.

    **Prevention:**
    1. One SKU per carton only
    2. Mixed sizes OK, mixed styles NOT OK
    3. Verify SKU matches all items before sealing
  </Accordion>

  <Accordion title="Sustainability non-compliance">
    **Cause:** Non-recyclable materials used.

    **Prevention:**
    1. Use recyclable poly bags only
    2. Paper void fill, not plastic
    3. Check supplier certifications
    4. Mobius loop symbol on all bags
  </Accordion>

  <Accordion title="Wrong carrier on ASN">
    **Cause:** ASN submitted before carrier confirmed.

    **Prevention:**
    1. Confirm carrier before submitting ASN
    2. If carrier changes, update ASN immediately
    3. Contact ASOS if ASN needs amendment
  </Accordion>
</AccordionGroup>

## Quick Reference Checklist

Before shipping to ASOS, verify:

- [ ] ASN submitted and acknowledged
- [ ] Carton labels: 100mm × 150mm, SSCC barcode
- [ ] One SKU per carton (mixed sizes OK)
- [ ] Each garment poly bagged with 3-language warning
- [ ] Poly bags have recycling symbol
- [ ] Carton weight ≤ 20 kg
- [ ] Commercial Invoice × 2 (GBP)
- [ ] Packing List with SSCC per carton
- [ ] ASN confirmation printed
- [ ] All SSCCs test-scanned
