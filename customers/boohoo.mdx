# Boohoo Requirements

This guide covers requirements for Boohoo Group brands including Boohoo, PrettyLittleThing (PLT), Nasty Gal, and Karen Millen.

<Warning>
Boohoo has strict compliance requirements. Non-compliant shipments may be rejected or result in chargebacks.
</Warning>

## Carton Label

### Label Layout

```
┌─────────────────────────────────────────┐
│  BOOHOO GROUP                           │
│  ─────────────────────────────────────  │
│  PO: 12345678                           │
│  Style: BGG27870-105                    │
│  Colour: Black                          │
│  Size: 10                               │
│  Qty: 12                                │
│  ─────────────────────────────────────  │
│  Carton: 1 of 15                        │
│  ─────────────────────────────────────  │
│  ║║║║║║║║║║║║║║║║║║║║║║║║║║║║║║║║║║║║║  │
│  DC BARCODE: 00123456789012345678       │
│  ─────────────────────────────────────  │
│  Weight: 12.5 kg                        │
│  Dimensions: 60 × 40 × 30 cm            │
└─────────────────────────────────────────┘
```

### Label Elements

| Element | Position | Format/Requirement |
|---------|----------|-------------------|
| Customer Logo | Top | Boohoo Group logo |
| PO Number | Upper section | 8-digit PO reference |
| Style Code | Upper section | Full style reference |
| Colour | Upper section | Colour name (not code) |
| Size | Upper section | UK size |
| Quantity | Upper section | Units in carton |
| Carton Number | Middle | "X of Y" format |
| DC Barcode | Lower section | 18-digit SSCC barcode |
| Barcode Number | Below barcode | Human-readable digits |
| Weight | Bottom | Gross weight in kg |
| Dimensions | Bottom | L × W × H in cm |

### Label Specifications

| Specification | Requirement |
|---------------|-------------|
| Size | 150mm × 100mm |
| Orientation | Portrait |
| Paper | White thermal, matte finish |
| Barcode Type | Code 128 |
| Barcode Height | Minimum 20mm |
| Quiet Zone | Minimum 5mm each side of barcode |
| Print Quality | Grade C or better |
| Placement | Two labels per carton (front and side) |

## Packing Requirements

### Carton Specifications

| Specification | Requirement |
|---------------|-------------|
| Maximum Weight | 15 kg gross |
| Maximum Length | 60 cm |
| Maximum Width | 40 cm |
| Maximum Height | 40 cm |
| Carton Material | Double-wall corrugated (min 300gsm) |
| Sealing | Clear tape, H-seal pattern |

### Packing Rules

<AccordionGroup>
  <Accordion title="Size Sorting">
    **Rule:** Single size per carton only.

    | Correct | Incorrect |
    |---------|-----------|
    | Carton 1: 12 × Size 10 | Carton 1: 6 × Size 10, 6 × Size 12 |
    | Carton 2: 12 × Size 12 | Mixed sizes in same carton |

    **Exception:** If order quantity is less than carton minimum, contact Boohoo for approval to mix sizes.
  </Accordion>

  <Accordion title="Poly Bagging">
    **Rule:** Each garment must be individually poly bagged.

    | Specification | Requirement |
    |---------------|-------------|
    | Bag Material | Clear polypropylene |
    | Bag Thickness | Minimum 25 micron |
    | Closure | Self-adhesive flap |
    | Warning | Suffocation warning printed on bag |
    | Size | Appropriate to garment (not oversized) |

    **Folding:** Garment folded neatly, facing visible through bag.
  </Accordion>

  <Accordion title="Sealing Method">
    **Rule:** H-seal pattern with clear tape.

    ```
    ┌───────────────┐
    │ ═══════════   │ ← Top seal
    │               │
    │       │       │ ← Center seal (both ends)
    │       │       │
    │ ═══════════   │ ← Bottom seal
    └───────────────┘
    ```

    | Specification | Requirement |
    |---------------|-------------|
    | Tape Width | 50mm clear packing tape |
    | Top Seal | Full width across top |
    | Bottom Seal | Full width across bottom |
    | Center Seals | Top to bottom on both ends |
    | Tape Quality | No lifting, bubbles, or wrinkles |
  </Accordion>

  <Accordion title="Carton Fill">
    **Rule:** Cartons should be appropriately filled.

    | Condition | Action |
    |-----------|--------|
    | Underfilled | Add void fill or use smaller carton |
    | Overfilled | Use larger carton, do not crush garments |
    | Void Fill | Kraft paper only (no peanuts or bubble wrap) |
  </Accordion>
</AccordionGroup>

## Required Documents

| Document | Required | Specifications |
|----------|----------|----------------|
| Commercial Invoice | Yes | 3 copies, GBP values |
| Packing List | Yes | Carton-level detail |
| Certificate of Origin | Yes | GSP Form A |
| Bill of Lading/AWB | Yes | Original + 2 copies |

### Commercial Invoice Requirements

| Field | Requirement |
|-------|-------------|
| Seller | Your company details |
| Buyer | Boohoo Group Limited |
| Currency | GBP |
| Incoterms | As per PO (usually FOB) |
| HS Codes | 6-digit HS codes for each style |
| Country of Origin | Pakistan |

### Packing List Requirements

| Field | Requirement |
|-------|-------------|
| Carton Numbers | Sequential (1 of X, 2 of X...) |
| Contents | Style, colour, size, quantity per carton |
| Weights | Gross and net weight per carton |
| Dimensions | L × W × H per carton |
| Totals | Total cartons, total pieces, total weight |

## Troubleshooting

<AccordionGroup>
  <Accordion title="Barcode won't scan at DC">
    **Common Causes:**
    - Print quality too low
    - Quiet zone insufficient
    - Barcode damaged or wrinkled

    **Prevention:**
    1. Use thermal printer with new ribbon
    2. Ensure 5mm minimum quiet zone
    3. Place label on flat carton surface
    4. Test scan before shipping
  </Accordion>

  <Accordion title="Carton rejected for weight">
    **Cause:** Carton exceeds 15 kg limit.

    **Prevention:**
    1. Weigh each carton before sealing
    2. Heavy items: fewer units per carton
    3. Use bathroom scale if industrial scale unavailable
    4. Record weights on packing list
  </Accordion>

  <Accordion title="Mixed sizes flagged">
    **Cause:** Multiple sizes found in single carton.

    **Prevention:**
    1. One size per carton only
    2. Label cartons during packing
    3. Double-check before sealing
    4. If mixing required, get written approval
  </Accordion>

  <Accordion title="Label peeling off">
    **Cause:** Poor adhesive or surface preparation.

    **Prevention:**
    1. Clean carton surface before applying
    2. Press firmly, especially edges
    3. Use proper label stock (not generic)
    4. Apply to flat surface, not over edges
  </Accordion>

  <Accordion title="Documents rejected">
    **Cause:** Missing information or wrong format.

    **Prevention:**
    1. Use Boohoo document templates
    2. Check all required fields completed
    3. Verify values match commercial invoice
    4. Ensure correct number of copies
  </Accordion>

  <Accordion title="Poly bag warning missing">
    **Cause:** Bags don't have suffocation warning.

    **Prevention:**
    1. Order bags with pre-printed warning
    2. Warning text must be visible
    3. Check bag specifications before ordering
    4. Keep stock of compliant bags
  </Accordion>
</AccordionGroup>

## Quick Reference Checklist

Before shipping to Boohoo, verify:

- [ ] Carton labels: 150mm × 100mm, Code 128 barcode
- [ ] Two labels per carton (front + side)
- [ ] Single size per carton
- [ ] Each garment poly bagged with warning
- [ ] Carton weight ≤ 15 kg
- [ ] H-seal tape pattern
- [ ] Commercial Invoice × 3 (GBP)
- [ ] Packing List with carton detail
- [ ] Certificate of Origin
- [ ] Barcodes test-scanned
