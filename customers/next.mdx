# Next Requirements

This guide covers requirements for Next PLC shipments, including their specific label format using A5 carton labels and EAN-13 barcodes.

<Warning>
Next uses EAN-13 barcodes (not Code 128). Ensure your labels use the correct barcode format.
</Warning>

| Maximum Height | 45 cm |

| Carton Color | Brown kraft (no printing) |
| Sealing | Clear tape, H-seal pattern |

### Packing Rules

<AccordionGroup>
  <Accordion title="Size Sorting">
    **Rule:** Single size per carton only.

    | Correct | Incorrect |
    |---------|-----------|
    | Carton 1: All Size 10 | Mixed sizes |
    | Carton 2: All Size 12 | Size 10 and 12 together |

    **Ratio packs:** If Next specifies ratio packs, follow their ratio exactly.

    | Ratio Pack Example | Contents |
    |--------------------|----------|
    | 1-2-2-1 (S-M-L-XL) | 1×S, 2×M, 2×L, 1×XL per pack |
  </Accordion>

  <Accordion title="Poly Bagging">
    **Rule:** Individual poly bagging required.

    | Specification | Requirement |
    |---------------|-------------|
    | Bag Material | Clear polypropylene |
    | Bag Thickness | Minimum 25 micron |
    | Closure | Fold-over with tape seal |
    | Warning | Child safety warning required |
    | Barcode Window | EAN must be visible through bag |

    **Presentation:** Garment folded with front facing up, visible through bag.
  </Accordion>

  <Accordion title="Sealing Method">
    **Rule:** H-seal with clear tape.

    ```
    ┌───────────────┐
    │ ═══════════   │ ← Top center seal
    │       │       │
    │       │       │ ← Vertical seals (edges)
    │       │       │
    │ ═══════════   │ ← Bottom center seal
    └───────────────┘
    ```

    | Specification | Requirement |
    |---------------|-------------|
    | Tape | 50mm clear packing tape |
    | Pattern | H-seal (top, bottom, both edges) |
    | Quality | No gaps, clean edges |
  </Accordion>

  <Accordion title="Carton Numbering">
    **Rule:** Sequential carton numbering required.

    | Field | Format |
    |-------|--------|
    | Carton Number | "X of Y" |
    | X | Current carton number |
    | Y | Total cartons in shipment |
    | Starting | Always start at 1 |

    **Example:** Shipment of 25 cartons → 1 of 25, 2 of 25... 25 of 25
  </Accordion>

  <Accordion title="Hanging Garments">
    **Rule:** Some items may require hanging.

    Check the PO for "hanging" designation. If hanging required:

    | Specification | Requirement |
    |---------------|-------------|
    | Hanger | Next-approved hanger only |
    | Cover | Clear poly garment cover |
    | Carton | GOH (garments on hangers) box |
    | Max per box | As specified on PO |
  </Accordion>
</AccordionGroup>

## EAN-13 Barcode Reference

Next uses EAN-13 barcodes. Ensure correct format:

### EAN-13 Structure

```
5 012345 678901
│ │      │
│ │      └── Item reference + check digit
│ └── Company prefix
└── Country code (5 = UK)
```

| Digits | Meaning |
|--------|---------|
| 1-3 | Country/region code |
| 4-7 | Company prefix |
| 8-12 | Item reference |
| 13 | Check digit (calculated) |

### Check Digit Calculation

The system auto-calculates check digits. If generating manually:

1. Sum odd positions (1,3,5,7,9,11) × 1
2. Sum even positions (2,4,6,8,10,12) × 3
3. Add both sums
4. Check digit = 10 - (sum mod 10)

<Info>
Always use system-generated barcodes to ensure correct check digits.
</Info>

## Troubleshooting

<AccordionGroup>
  <Accordion title="EAN barcode not scanning">
    **Common Causes:**
    - Wrong barcode type (Code 128 instead of EAN-13)
    - Incorrect check digit
    - Print quality issues

    **Prevention:**
    1. Verify barcode is EAN-13 format
    2. Use system-generated barcodes
    3. Print on laser printer
    4. Test scan before shipping
  </Accordion>

  <Accordion title="Label size rejected">
    **Cause:** Using wrong label size.

    **Prevention:**
    1. Next requires A5 (148mm × 210mm)
    2. Use correct template
    3. Verify printer settings before printing batch
  </Accordion>

  <Accordion title="Carton overweight">
    **Cause:** Exceeds 18 kg limit.

    **Prevention:**
    1. Weigh each carton
    2. Leather jackets: fewer per carton
    3. Record weights accurately
    4. Leave margin (aim for 17 kg max)
  </Accordion>

  <Accordion title="Mixed sizes flagged">
    **Cause:** Multiple sizes in same carton.

    **Prevention:**
    1. Single size per carton
    2. Unless ratio pack specified
    3. Check PO for packing instructions
    4. Label cartons during packing
  </Accordion>

  <Accordion title="Wrong hanger used">
    **Cause:** Non-approved hanger for hanging garments.

    **Prevention:**
    1. Check if hanging required (PO)
    2. Use only Next-approved hangers
    3. Order hangers from approved supplier
    4. Verify hanger type before production
  </Accordion>

  <Accordion title="Style number mismatch">
    **Cause:** Label shows different style than contents.

    **Prevention:**
    1. Verify style number from PO
    2. Cross-check product labels
    3. Double-check before sealing carton
    4. One style per carton only
  </Accordion>
</AccordionGroup>

## Quick Reference Checklist

Before shipping to Next, verify:

- [ ] Carton labels: A5 size (148mm × 210mm)
- [ ] Barcode type: EAN-13 (not Code 128)
- [ ] One label per carton (largest face)
- [ ] Single size per carton (unless ratio pack)
- [ ] Each garment poly bagged
- [ ] EAN visible through poly bag
- [ ] Carton weight ≤ 18 kg
- [ ] H-seal tape pattern
- [ ] Commercial Invoice × 3 (GBP)
- [ ] Packing List with size breakdown
- [ ] Sequential carton numbering (1 of X, 2 of X...)
- [ ] All barcodes test-scanned
